# Yet Another Pod File Agent - 一个简易的边车程序，可以使你在清凉的夏夜进行无忧的安眠

## 项目描述

这是一个简单的file agent，启动后会监听8080端口并向外提供一个基于`HTTP1.1`的`protobuf gRPC connect`服务，它可以做到：
1. 获取 指定根目录 及其子目录的所有数据信息
2. 接受上传数据至 指定根目录 及其子目录
3. 下载 指定根目录 及其子目录的文件
4. 移动，复制或删除 指定根目录 里面的文件夹或文件

**特别注意：没有鉴权！这个项目最适合使用的方法是作为主pod的辅助pod使用！！！**

## 食用方式

添加在你 k8s 的 statefulset 里面，然后读读API文档，拉下来这个库，dial就好了(WIP...)

容器镜像地址：`ghcr.io/chi-net/yapfa`
